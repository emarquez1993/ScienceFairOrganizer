Project Instructions:
1. Before starting the program, make sure that the judges, projects, and output text files are in the root of the project folder or in the same folder as the source code files.
2. In the GUI, first use the buttons to select the judges and projects text files.
3. Next, use the button that will check to see if the text files are in the correct folders as specified in step one.
4. Use the right-most button to select the output text file. The text box will let you know if the judge assignment was successfully written to the output file.
